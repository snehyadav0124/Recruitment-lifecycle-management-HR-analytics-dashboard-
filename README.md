# Recruitment-lifecycle-management-HR-analytics-dashboard-
This project demonstrates an end to end recruitment and onboarding management system build in Microsoft Excel 
It covers the complete hiring lifecycle—from candidate application tracking and interview coordination to offer management, onboarding, and recruitment analytics reporting.
The project showcases practical HR operations, recruitment coordination, data management, reporting, and dashboard creation using Excel functions, data validation, pivot tables, charts, and HR metrics.
Key Skills Demonstrated
Recruitment Coordination
Candidate Database Management
Interview Scheduling & Tracking
Offer Management
Employee Onboarding Tracking
HR Analytics & Reporting
Excel Dashboard Development
Data Validation & Data Management
KPI Tracking
Pivot Tables & Charts
